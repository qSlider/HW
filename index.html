git config - настройка конфигурации
git config --global user.email blabal@bla - устанавливает адресс электронной почты которая отображается в информации
git init - для запуска репозитория
git status показывает состояния файлов в рабочем каталоге
git add добавляет содержимое рабочего каталога в индекс
git commit – это команда Git для записи индексированных изменений в репозиторий.
git log перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке
git checkout - файлы копирует файлы со сцены в рабочую директорию
git branch - она умеет перечислять ваши ветки, создавать новые, удалять и переименовывать их
git push - команда создает локальную ветку в репозитории назначения
git pull — это, по сути, команда git fetch, после которой сразу же следует git merge
git merge используется для слияния одной или нескольких веток в текущую
git clone — репозитория и создания егоп клона
